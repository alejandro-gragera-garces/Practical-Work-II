# Practical-Work-II for Fundamentals of Computer Engineering. 
First and foremost, the 2nd practial work of the course consisted off the creation of a web page. It was specified that the web page had to include the following html files: about.html (self-description), contact.html (allow easy contact method for user), degree.html.    (showcases degree plan), fce.html (describes the contents of the course of "Fundamentals of computer Engineering"), net.html (links partners with similar projects) and index.html (1st page introduction). All these files are displayed with a single css style <link rel="stylesheet" href="../css/styles.css">.  
From my experience, completing this project was a hard, tedious task. I had many hitches during my developement since I lost a nearly complete project and had to start over. Moreover, looking back I'd say it has eventually grown on me and even become an enjoyable proyect I may repeat to create my own webs. I can confidently I have some unpolished (very unpolished) skills now with CSS that have let me customize my web much more in a shorter time than I expected.
